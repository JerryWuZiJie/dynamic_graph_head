# Dynamic Graph Head (dgh)

## What is this

Utils and infrastructure for using Dynamic Graph Head in simulation and on the real robot.

## Installation



```
$ treep --seq-clone DYNAMIC_GRAPH_HEAD

# From within the repo
$ pip install .
```
